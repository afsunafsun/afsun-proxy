# Архитектура v2

## Компоненты

```
┌─────────────────────────────────────────────────────────────┐
│  VPS                                                        │
│  ┌──────────┐   ┌──────────┐   ┌─────────────────────────┐  │
│  │ Caddy    │──►│ 3X-UI    │──►│ Xray (host)             │  │
│  │ :8443    │   │ :2053 loc│   │ :443 VLESS Reality      │  │
│  └──────────┘   └──────────┘   │ :2087 VLESS alt (opt)   │  │
│                                 │ :10808 SOCKS            │  │
│  ┌──────────────────────────┐  └─────────────────────────┘  │
│  │ afsun-hysteria2 (systemd)│  UDP :8444 — official HY2     │
│  └──────────────────────────┘                               │
└─────────────────────────────────────────────────────────────┘
```

- **3X-UI** — панель, пользователи, inbounds, подписки, SQLite в `/var/lib/afsun-proxy/x-ui/db`
- **Xray** — внутри 3X-UI, `network_mode: host`
- **Caddy** — TLS для панели и ACME; порт 443 занимает Xray
- **Hysteria2 (official)** — отдельный systemd-сервис `afsun-hysteria2`, конфиг `/var/lib/afsun-proxy/hysteria/server.yaml`, LE-сертификат от Caddy. Строка HY2 в подписке — через external link в БД, **не** через inbound панели

## Порты

| Порт | Сервис |
|------|--------|
| 443 | VLESS Reality (клиенты) |
| 8443 | HTTPS панель + подписка |
| 80 | ACME |
| 10808 | SOCKS (опционально, интеграции) |
| 8444/udp | Hysteria2 official (опционально) |
| 2087 | VLESS alt (опционально) |

Панель 3X-UI: **127.0.0.1:2053** — наружу не открывать.

## Отличие от v1 (Marzban)

| | v1 Marzban | v2 3X-UI |
|---|------------|----------|
| БД | MariaDB | SQLite в 3X-UI |
| SOCKS | костыль + webhook | нативно |
| Hysteria2 | нет | official systemd + external link |
| Сложность | высокая | ниже |

## Что не использовать

| Компонент | Причина |
|-----------|---------|
| Xray inbound «Hysteria2» в панели | Несовместим с Streisand/Hiddify (Xray #5921) |
| VLESS на «случайных» портах без теста | Может резаться DPI; используйте alt port скрипт |
