# Acceptable Use

Используя **afsun-proxy**, вы соглашаетесь с правилами ниже.

## Разрешено

- Развёртывание на **своём** VPS для **своих** пользователей и интеграций.
- Выдача доступа сотрудникам, клиентам по договору — при соблюдении местного законодательства.
- SOCKS для **своих** приложений (боты, автоматизация, мониторинг).

## Запрещено

- Предоставление анонимного массового VPN «для всех» без учёта правовых требований.
- Хранение паролей и ключей в публичном git-репозитории.
- Использование для спама, атак, сканирования, нарушения ToS сторонних сервисов.

## Администратор обязан

- Вести учёт клиентов (email / subId в панели).
- Отключать скомпрометированные учётные записи.
- Открывать в UFW **только** нужные порты.
- Обновлять Docker-образы и следить за CVE.

## Hysteria2 и подписка

Официальный Hysteria2 работает **вне** inbound панели (systemd). При добавлении клиента обновите конфиг HY2 — см. [OPERATIONS.md](OPERATIONS.md).
