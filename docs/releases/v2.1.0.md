## Что вошло в v2.1.0

### Установка

- **`bootstrap.sh`** — одна команда: панель, UFW, VLESS Reality :443, подписка
- **`install.sh`** — Docker, geofiles, Caddy из шаблона

### Протоколы

- VLESS + Reality (авто через API 3X-UI)
- Official Hysteria2 (systemd, не Xray inbound)
- SOCKS, alt VLESS port (2087)

### Эксплуатация

- `sync-ufw-inbounds.sh` — порты inbound'ов в UFW (панель сама не открывает)
- `fix-reality-target.sh` — поле **Цель** для 3X-UI 3.4.0

### Документация

- DEPLOY, OPERATIONS, INBOUND-SETUP, TROUBLESHOOTING, SECURITY, DISCLAIMER

### Быстрый старт

```bash
git clone https://github.com/afsunafsun/afsun-proxy.git /opt/afsun-proxy
cd /opt/afsun-proxy && git checkout v2.1.0
cp .env.example .env && nano .env
sudo bash scripts/install.sh && sudo bash scripts/bootstrap.sh
```
